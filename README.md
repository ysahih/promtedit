# PromptEdit — AI-Powered Video Editing Templates & Tools

A modern homepage redesign for [promptedit.com](https://promptedit.com) — built as a job application portfolio piece demonstrating front-end design and development skills.

**Live Demo → [promptedit.vercel.app](https://promptedit.vercel.app)**

---

## What Was Built

A full-page marketing site that sells PromptEdit's marketplace of video editing templates and pay-per-use AI tools. Designed and implemented **directly in Tailwind + HTML** — no Figma step.

### Sections
1. **Navigation** — Sticky transparent header with blur effect, mobile hamburger menu
2. **Hero** — Full-screen cinematic opener with typing animation and animated stat counters
3. **AI Tools** — Glassmorphism cards for all 6 AI models
4. **Templates Marketplace** — Category pills, preview grid, supported editor icons
5. **How It Works** — Clean 3-step workflow
6. **Editor Extensions** — DaVinci Resolve & Premiere Pro plugin showcase
7. **Pricing** — Free vs Unlimited comparison cards
8. **Footer** — Multi-column layout with social icons

| Tool | Purpose |
